# HOHQMesh runtime

Bundled native HOHQMesh 1.5.5 executables and CompilerSupportLibraries 1.5.1 runtime libraries from JuliaBinaryWrappers.

Sources:
- https://github.com/trixi-framework/HOHQMesh
- https://github.com/JuliaBinaryWrappers/HOHQMesh_jll.jl
- https://github.com/JuliaBinaryWrappers/CompilerSupportLibraries_jll.jl

HOHQMesh is MIT licensed. CompilerSupportLibraries packages the GCC runtime libraries needed by the executable.
